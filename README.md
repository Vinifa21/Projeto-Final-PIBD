#  Entrega final do Trabalho da disciplina de Projeto e Implementação de Banco de Dados

## Tema: Recursos Humanos: Professores, Técnicos, Folha de Pagamento, Departamentos, Cargos.

## Como rodar o projeto:

Garantir que exista um banco de dados PostgreSQL com o nome "faculdade" na sua máquina     
Garantir que suas credenciais sejam as mesmas das que estão na função "obter_conexao" do arquivo "database.py"     
Instalar a biblioteca "psycopg2"     
Executar o arquivo app.py
